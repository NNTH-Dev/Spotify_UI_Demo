## Install Module
--> yarn install

## Build Enviroment IOS 
--> yarn pod-install

## Build Enviroment Android 
- create file local.properties with content 'sdk.dir = /Users/UserName/Library/Android/sdk'
--> cd android && ./gradlew clean

## Build App
--> yarn start
